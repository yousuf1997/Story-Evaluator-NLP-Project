BerTest.py file is the driver code file. However, in order to visualize the result we need to follow the steps written below before we can execute the driver code file. Please follow the steps to install the necessary modules.

1. Install torch. Go to the terminal of your IDE if you are using VS Code, Pycharm or IntelliJ and type the following command " pip install torch --user"
2. Install matplotlib. Use the same command as given above
3. Install transformers.
4. Install scipy.
5. Now follow the commands as given
	"pip3 install https://github.com/explosion/spacymodels/releases/download/en_core_web_sm-2.2.0/en_core_web_sm-2.2.0.tar.gz"
Once you have pasted and run the above text paste this 
"python -m spacy download en_core_web_sm"

Now you have installed all the required modules to run the implemented code and can check how the logic responds to different stories along with the elements of hint and foreshadowing in them.
